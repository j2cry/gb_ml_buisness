Catch me if you can

This model allows you to detect an intruder by analyzing web sessions. The original Kaggle competition is here.
And here you can see the source code.

How to use

Send to this URL a post request contains sample data. It can contain either one or many samples. Sample data must have the next structure:
{column_name: {object_id: value, ... }, ... }
The best way to get this is the pandas.to_dict() method.
The button below sends randomized samples collected from original train data.
